# Incident & Troubleshooting Logs

This directory contains **chronological incident logs** documenting
hardware, firmware, operating system, networking, and service-level
issues encountered while building and operating this homelab.

Each log represents a **single problem or investigation**, recorded to
capture:
- Symptoms and observed failures
- Diagnostic steps taken
- Hypotheses and reasoning
- Root cause (when identified)
- Final outcome or resolution

## Scope

Logs may include (but are not limited to):
- POST and firmware issues
- Hardware compatibility problems
- Proxmox configuration and networking errors
- VM and container failures
- Storage, performance, or stability issues
- Security and access-related incidents

## Conventions

- One log file per issue
- Logs are written after or during the incident
